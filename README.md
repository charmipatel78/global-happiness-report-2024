World Happiness Report 2024 – Data Analysis Project:

Project Overview:
This project focuses on analyzing the World Happiness Report 2024 dataset to understand the factors that influence happiness scores across different countries.
The goal of this analysis is to clean the data, explore key variables, and extract meaningful insights using data analytics tools.

---

  Dataset Information:
- Dataset Name:World Happiness Report 2024
- Source: Kaggle / World Happiness Report
- Data Type: CSV file
- Key Columns:
  - Country name
  - Happiness score
  - GDP per capita
  - Social support
  - Healthy life expectancy
  - Freedom to make life choices
  - Perceptions of corruption

---

Tools Used
- Excel:
  - Data cleaning
  - Sorting & filtering
  - Pivot tables
  - Charts & visualizations
- Python:
  - Pandas for data analysis
  - NumPy for calculations
  - Exploratory Data Analysis (EDA)

---

  Analysis Performed:
- Cleaned missing and inconsistent data
- Compared happiness scores across countries
- Analyzed relationship between GDP and happiness
- Identified top and bottom countries by happiness score
- Studied impact of social support and life expectancy on happiness

---

  Key Insights:
- Countries with higher GDP per capita generally show higher happiness scores
- Social support plays a major role in overall happiness
- Higher healthy life expectancy is strongly linked to higher happiness
- Nordic countries consistently rank among the happiest countries
- Countries with higher perceived corruption tend to have lower happiness scores

---

Conclusion:
The analysis shows that happiness is influenced by multiple socio-economic factors, not just income. Social support, health, and freedom significantly contribute to a country’s happiness level.
This project helped strengthen my skills in data cleaning, analysis, and deriving insights from real-world datasets.
